# HDW — Portfolio

Professional TUI-style portfolio website for Hafiz Daffa W.

## Tech Stack

| Category | Technology |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript 5 |
| UI Library | React 19 |
| Styling | Tailwind CSS 4 |
| Component Kit | shadcn/ui (base-nova) |
| Animation | Motion |
| Code Highlighting | prism-react-renderer (Night Owl) |
| Icons | Lucide React |
| Fonts | Geist Mono (via next/font) |

## Design System

- **Theme:** Dark TUI (Terminal User Interface)
- **Colors:** Green (`#22c55e`), Cyan (`#06b6d4`), Orange (`#f97316`)
- **Typography:** Monospace (Geist Mono)
- **Borders:** Subtle `rgba(255,255,255,0.06)`, 2px radius
- **Panels:** Terminal-style with headers and dividers

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Sections

1. **Hero** — Name, role, CTAs (GitHub, Email, Instagram)
2. **About** — Bio, current exploration, stats panel
3. **Experience** — Work history with tech tags
4. **Sulfur++** — Featured project with code block
5. **Tech Stack** — Categorized skills with proficiency levels
6. **Projects** — Project cards with tags and status
7. **Timeline** — Achievements & competitions log
8. **Tinker's** — Community club showcase
9. **Footer** — ASCII art + social links
